1-Reverse a String: Take a string as input and print it backwards (e.g., "Python" becomes "nohtyP").

2-Vowel Counter: Write a program to count how many vowels (a, e, i, o, u) are in a given sentence.

3-Palindrome Check: Check if a string is a palindrome (reads the same forward and backward, like "madam").

4-Remove Whitespace: Take a string with extra spaces at the beginning and end, and remove them using string methods.

5-Word Counter: Count how many words are in a sentence by splitting the string.

6-Capitalize Words: Take a sentence like "hello world" and convert it to "Hello World" (First letter of each word capital).

7-Find and Replace: In the string "I like apples", replace "apples" with "bananas" using code.

8-Character Frequency: Count how many times the letter 'a' appears in a given string.

9-Anagram Check: Take two strings and check if they are anagrams (contain the same characters in different order, like "listen" and "silent").

10-Password Validator: Check if a string (password) has at least 8 characters, one uppercase letter, and one number.
